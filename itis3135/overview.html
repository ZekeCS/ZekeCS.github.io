Project Overview

Pro Plus Consulting is a growing HR consulting firm. This site is designed to showcase their past projects, provide estimates, and facilitate new client engagement. It includes a scrollable project gallery, dynamic estimation tools, and secure login portals for current clients to manage communication and ongoing projects.

Site Map

Website Pages & Functionality

  • Home/About Us: Company mission, services, and navigation hub.
  • Login: Secure login for clients and employees with validation and redirect functionality.
  • Create Account: Client registration via a company-provided code with password matching validation.
  • Hire Us: Form with validation for new client contact requests.
  • Get an Estimate: Interactive JS-based estimate tool based on client needs and task variables.
  • Client Portal: Messaging system, scheduling modals, past work tracking, and communication tools.

Dynamic Functionality

  • Dynamic headers/footers updated via JavaScript.
  • Login/Create Account validation and redirection.
  • Estimate calculator with display results in new div.
